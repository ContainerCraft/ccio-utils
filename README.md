# p1-gotools
golang dev tool for p1 init script utilities

## Development Environment:
  1. Clone Repo
```
 git clone git@github.com:containercraft/p1-gotools.git && cd p1-gotools    
```
  2. Launch development container
>   (Requires podman installed)
```
 . tools/podman.sh
```
  3. Run program
```
 go run main.go
```
